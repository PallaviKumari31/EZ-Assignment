# Varnan Films Website Clone For EZ


This repository contains the source code for the website of Varnan Films. It's a visually rich, single-page application built with React and Vite, designed to showcase their work in film production, branding, and art curation. The site features smooth animations, a responsive layout, and a functional contact form.

## Features

-   **Responsive Design:** Fully adaptive layout for optimal viewing on desktops, tablets, and mobile devices.
-   **Rich Animations:** Engaging user experience with animations and transitions powered by Framer Motion.
-   **Component-Based Architecture:** Built with React, featuring well-structured components for different sections like a hero showcase, service overview, client montage, and video reel.
-   **Interactive Contact Form:** A complete contact form with client-side validation and submission to a backend API, including a success notification popup.
-   **Modern Tooling:** Fast development and build process using Vite and utility-first styling with Tailwind CSS.

## Tech Stack

-   **Frontend:** React
-   **Build Tool:** Vite
-   **Styling:** Tailwind CSS
-   **Animation:** Framer Motion
-   **Icons:** Lucide React

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have Node.js and npm (or a compatible package manager like yarn or pnpm) installed on your system.

### Installation

1.  Clone the repository to your local machine:
    ```sh
    git clone https://github.com/pallavikumari31/ez-assignment.git
    ```

2.  Navigate to the project directory:
    ```sh
    cd ez-assignment
    ```

3.  Install the required dependencies:
    ```sh
    npm install
    ```

4.  Start the development server:
    ```sh
    npm run dev
    ```

The application will now be running on a local server, typically `http://localhost:5173`. Open this URL in your browser to view the site.
